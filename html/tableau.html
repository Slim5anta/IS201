Global Superstore Tableau



This tableua dashboard features four worksheets with different information regarding profit. To see the data on each sheet, be sure to click on the tab in the top left corner.

Table 1 (Profit by State) shows data on how each state in the United States of America is fairing in regard to profit compared to each other.

Table 2 (California Profit by Month) shows data on how California profits have been over time.

Table 3 (California Profit per Category) shows the breakdown of each product category and how profits are for California within those categories.

Table 4 (Forecast of Office Supplies in California) shows the previous data as well as the projected data in regard to profits for office supplies for California.
